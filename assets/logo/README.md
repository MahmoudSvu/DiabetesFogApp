# لوغو التطبيق

## الموقع:
ضع ملفات اللوغو هنا:
- `logo.png` - اللوغو الرئيسي (512x512 بكسل)
- `logo_icon.png` - أيقونة صغيرة (192x192 بكسل)

## المتطلبات:
- الخلفية: شفافة أو بيضاء
- الألوان: متباينة مع اللون الرئيسي #041E76
- الشكل: دائري أو مربع بزوايا دائرية
- الحجم: 512x512 للوغو الرئيسي، 192x192 للأيقونة

## التصميم المقترح:
- أيقونة قلب مع موجة (monitor_heart icon style)
- لون الأيقونة: أبيض أو فاتح على خلفية #041E76
- أو أيقونة قلب بلون #041E76 على خلفية بيضاء

## بعد إضافة الملفات:
سيتم استخدام اللوغو تلقائياً في:
- شاشة الترحيب
- AppBar في جميع الصفحات
- أيقونة التطبيق في قائمة التطبيقات
